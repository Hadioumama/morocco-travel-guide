# Land of Morocco 🇲🇦

A modern travel guide website to explore Morocco’s top destinations, culture, and travel experiences.

## 🌍 Overview
Land of Morocco is a travel guide platform that showcases the most beautiful destinations across Morocco including imperial cities, deserts, mountains, and historical landmarks.
In each city, the user can find the **best restaurants**, exciting **adventures**, and some **extra information** to plan their trip.
## ✨ Features
- Beautiful homepage with slider
- Destinations section
- itineraries section
- about section
- contact section
- Responsive design
